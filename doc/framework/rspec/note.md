# 2章 Rspecのセットアップ
* RspecはデフォルトではRailsに含まれていないため、Bundlerを使ってインストールする必要がある
* rspec-railsはrspec-coreとその他の独立したgemが含まれている。
* Rails以外のアプリケーションでRspecを使いたい場合は、個別にインストールする必要がある
